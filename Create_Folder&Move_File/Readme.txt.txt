Chương trình được chạy bằng CMD
Vào star gõ cmd 
Bấm mở cmd
Chọn đường dẫn đế thư mục Create_Folder&Move_File bằng cú pháp
chuột phải chọn properties phần location trong tab hiện ra sẽ là đường dẫn
Trong CMD gõ: cd <đường dẫn>
Chạy chương trình với cú pháp: 
python Create_Folder&Move_File.py SourcesFolder <Số bắt đầu:(ví dụ: 50)>
hoặc
python3 Create_Folder&Move_File.py SourcesFolder <Số bắt đầu:(ví dụ: 50)>
hoặc
Create_Folder&Move_File.py SourcesFolder <Số bắt đầu:(ví dụ: 50)>
